# imovie-node	
一个简单的node学习项目
主要用到：
   bootstrap
   jquery
   express
   jade
   mongodb
   underscore
	
	主页：'localhost:3000'
        编辑电影后台:'localhost:3000/admin/movie'
        更新电影:'localhost:3000/admin/update/:id';
        登录:'localhost:3000/signin';
        注册:'localhost:3000/signup';
        后台电影列表:'localhost:3000/admin/list';
	
项目运行：
   1.安装mongodb
   2.连接mongodb  cmd   mongod.exe --dbpath d:\data\db
   3.npm install 
   4.node app
